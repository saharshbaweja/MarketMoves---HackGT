Project Description

This project is a financial news-based stock evaluator, providing real-time stock sentiment analysis and generating a recommendation of Buy, Hold, or Sell based on sentiment scores. It uses technical indicators such as Simple Moving Average (SMA) and Exponential Moving Average (EMA) to rate stocks.

Features

	•	Sentiment analysis on stock-related news articles
	•	Technical indicators (SMA, EMA, etc.)
	•	Simple HTML-based frontend connected to a Flask backend

Technologies Used

	•	Frontend: HTML, CSS, JavaScript
	•	Backend: Flask, Python
	•	APIs: News API, yfinance, sentiment analysis (VADER)
	•	Development Tools: virtualenv, Git

Future Enhancements

	•	Add reinforcement learning algorithms to improve stock prediction accuracy
	•	Implement more technical indicators such as MACD, RSI
	•	Expand sentiment analysis by integrating multiple data sources
